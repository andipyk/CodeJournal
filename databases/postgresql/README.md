# Postgresql Database

psql commands
=============

go to psql command line:
	sudo -u postgres psql

## show databases:
	\l

## connect to database # to create relations
	
	\c database_name

	database_name# \dt to list all tables
	
	SELECT * FROM table_name;  # to select the table 

## user list:
    
    \du
    
## table list:
    
    \d
