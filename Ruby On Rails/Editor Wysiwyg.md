# WYSIWYG HTML Editor

[Rails Froala WYSIWYG HTML Editor](https://github.com/froala/wysiwyg-rails)

Add this to your Gemfile:

	gem "wysiwyg-rails"

	gem "font-awesome-rails"


	*= require froala_editor.min.css
	*= require froala_style.min.css
	*= require font-awesome
